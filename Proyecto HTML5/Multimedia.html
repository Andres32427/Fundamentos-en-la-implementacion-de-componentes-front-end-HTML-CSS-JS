< area> Se utiliza dentro de un mapa de imagen para poder definir regiones en especificas en una imagen que son clicable. alt: Este proporciona un texto alternativo en el area.
coords: Define las coordenadas de la region clicable dentro de la imagen.
shape: Esta especifica la forma de la region clicable (rect, circle, poly, default).
href: Especifica la URL a la que se dirigira el usuario.
target: Se define donde se abrira el enlace.
rel: Define la relacion entre el documento actual y el documento enlazado.

< canvas> Se utiliza para dibujar gráficos, por script, en el aire. (con JavaScript). width: Especifica el ancho del area de dibujo en pixeles. valor por defecto: 300px.
height: Especifica la altura del area de dibujo en pixeles. valor por defecto: 150px.
id: Define un identificador unico para el elemento.
class: Permite asignar clases CSS.
style: Permite aplicar estilos CSS directamente.

Mi Dibujando en Canvas
< embed> Se utiliza para insertar contenido multimedia o aplicaciones externas dentro de una página web. src: URL del recurso multimedia.
type: tipo MIME del recurso.
width: Ancho del contenido incrustado.
height: Alto del contenido incrustado.
< iframe> Se utiliza para incrustar otro documento HTML dentro del documento actual, permitiendo la visualizacion del contenido externo como videos, mapas, o páginas web completas en un marco dentro de la página. src: Define la URL del documento a incrustar.
width: Especifica el ancho del iframe.
height: Especifica la altura del iframe.
frameborder: Controla la visibilidad del borde.
allow: permite ciertas funcionalidades como fullscreen.
sandbox: Aplica restricciones de seguridad.

Incrustando una Página Web
< img> Se utiliza para incrustar imágenes en una página web. Es fundamental para añadir elementos visuales al contenido. src: Define la URL de la imagen.
alt: Proporciona texto alternativo.
width: Especifica el ancho de la imagen.
height: Especifica la altura de la imagen.
title: Informacion adicional visible al pasar el cursor.
loading: Controla la carga de la imagen (lazy, eager).
< map> Se define un mapa de imagen, que es una imagen con áreas interactivas. Cada área dentro del mapa de imagen se define con la etiqueta < area>, que puede ser un enlace o un elemento interactivo. name: Define el nombre del mapa de imagen, usado para asociar con la imagen mediante el atributo < usemap>
< object> Se utiliza para incrustar un recurso externo, como un video, una imagen, un documento o una aplicación. data: URL del recurso a incrustar.
type: Tipo de contenido del recurso.
width: Ancho del objeto.
height: Altura del objeto.
name: Nombre del objeto.
form: Formulario al que pertenece el objeto.
usemap: Asocia un mapa de imagen.
Atributos Globales: Como id, class, style, etc.
< param> se utiliza dentro de la etiqueta < object> para definir parámetros que afectan el comportamiento o la visualización del objeto incrustado. name: Define el nombre del parametro.
value: Especifica el valor del parametro.
< source> Se utiliza para especificar uno de los posibles recursos de medios que se puede reproducir con un elemento < audio>, < video>, o < picture>. src: Especifica la URL del archivo multimedia.
type: Define el tipo de medio o formato del archivo.
media: Un atributo opcional que define las condiciones bajo las cuales se utiliza el recurso. Funciona como una consulta de medios CSS.
< track> Se utiliza para proporcionar pistas de texto (como subtítulos, descripciones de audio, o capítulos) para contenido multimedia. src: URL de la pista de texto.
kind: Tipo de pista (subtitulos).
srclang: Idioma de la pista.
label: Titulo de la pista.
default: Define si esta pista es la predeterminada.
< video> Se utiliza para incrustar videos en documentos HTML. Esta etiqueta permite incluir múltiples fuentes de video, proporcionando soporte para diferentes formatos y navegadores. src: Especifica la URL del archivo de video.
controls: Muestra los controles del video como play, pausa, etc.
autoplay: El video comienza a reproducirse automáticamente cuando se carga la página.
loop: Hace que el video se repita después de que finaliza.
muted: El video se reproduce sin sonido.
poster: Especifica una imagen que se muestra mientras se carga el video o hasta que el usuario lo reproduce.
width: Especifica el ancho.
height: Especifica la altura.
